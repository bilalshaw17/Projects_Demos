# Airplane Game ✈️

**Airplane Game** is a simple 3D flight game developed in **Unity** where the player controls an airplane, navigates through the environment, and avoids obstacles to reach the destination safely.

### Features

* Smooth airplane controls.
* Simple 3D environment.
* Obstacle avoidance gameplay.
* Basic flight mechanics.
* Easy-to-play and beginner-friendly design.

### Technologies Used

* Unity Engine
* C#
* Unity Physics

### Learning Outcomes

This project enhanced my understanding of Unity game development, aircraft movement, player controls, collision detection, and implementing core gameplay mechanics using C#.
